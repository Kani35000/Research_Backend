ABOUT RESEARCH

Building an Application that Visualizing newsworthiness of tweets using ML


It is mechanism to capture a user’s received tweets (from followers/pay loads) for the day and create a small dataset (tweets ) and run the dataset on the Machine Learning classifier and then visualize the tweets which are news and which are not news for the particular user,

Overall project expectation:
Users logins via the web browser and sees news / not news tweets (everything happens under the hood)
Tasks:
Phase 1:
Infrastructure creation: Create an infrastructure (through twitter api) to allow twitter users to log in via a web browser extension
OUTPUT - Django app that can allow twitter login
This will be tested using 5 student twitter users accounts.


Phase 2: Data mining: Once log in - users daily received tweets (-last 24 hours - from followers etc) is mined into a dataset
Individual tweets datasets (for each user) - needs to be cleaned and processed ready to be run on the classifier
OUTPUT: a file with cleaned tweets received for a logged in user)

Classifier labels each tweet as news/not news
OUTPUT: labeled tweets showing news /not news


Phase 3: Classified tweets are visualized for the user on the browser extension
Newsworthy tweets should appear as priority tweets at the top

All other tweets (non news should be labeled at the bottom of the user’s feed
High priority tweets could be displayed for the user etc.

Popup notification for tweets classified as news.
